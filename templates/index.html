Getting Started

Setup Instructions

  1. Create a Telegram Bot

    Use BotFather (@BotFather) on Telegram to create a new bot and get your API token.

  2. Set Environment Variables

    Set the following environment variables in your Replit:

    BOT_TOKEN, API_ID, API_HASH
  3. Start the Bot

    Use the "Start Bot" button above to start the bot service.

  4. Configure on Telegram

    Start a chat with your bot on Telegram and use the /settings command to add source and destination channels.

  5. Start Forwarding

    Use the /forward command to begin forwarding messages from your source to destination.
