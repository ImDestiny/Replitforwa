How to Use
  1. Start the bot and login with your Telegram API credentials
  2. Add source channels/groups from which you want to forward messages
  3. Add destination channels/groups where messages will be forwarded to
  4. Optionally set the last message ID or link to start forwarding from
  5. Start the forwarding process and monitor progress

You can also interact with the bot directly on Telegram using commands like /start, /forward, /tasks, and /settings.
